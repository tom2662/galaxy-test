# galaxy-test
galaxy merchant trading apps based on go


# Installation

 - pull from repository
   git@github.com:tom2662/galaxy-test.git

 - build image
    docker build -t galaxy:1.0 .

 - run
    docker run -t -i -p  8000:8000 galaxy:1.0

# Usage

  - API Endpoint 
    localhost:8000/converter/
